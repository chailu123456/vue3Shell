# vue3-2

## Project main
```
使用node进行git操作，运行npm run shell,进行一系列git add,git commit,git push
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### 快速推送代码
```
npm run shell 你的commit说明
```
